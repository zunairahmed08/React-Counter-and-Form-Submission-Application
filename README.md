Project Title

React Counter and Form Submission Application

Introduction

This project is a simple React-based application designed to demonstrate the concepts of state management, event handling, form validation, list rendering, and the use of browser localStorage. The application contains a counter feature and a form submission system that stores and displays user-entered data.

Features
1. Counter System

Increment, decrement, and reset functionality

Counter value is stored in browser localStorage

Keyboard support:

ArrowUp to increase

ArrowDown to decrease

R key to reset

2. Form Handling

Users can enter name, email, and notes

Validates empty fields

Email format validation

Displays error messages for invalid input

3. Submissions Section

Shows all submitted data

Saves submissions in localStorage

Allows deletion of any submitted record

Technologies Used

React.js

JavaScript (ES6+)

React Hooks (useState, useEffect)

LocalStorage API

CSS for styling

Project Structure:
src/
│── App.jsx
│── App.css
│── index.css
│── main.jsx
└── assets/

How to Run the Project
Step 1: Install the required packages
npm install

Step 2: Start the development server
npm run dev


Step 3: Open the application in the browser
http://localhost:5173/
